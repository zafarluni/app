# Ayla Home Privacy Policy

Last updated: 2026-01-28

Ayla Home is a household coordination application designed to help people living together manage shared responsibilities.

## Information We Collect

When you sign in using Google, we collect only basic profile information provided by Google:
- Name
- Email address
- Profile photo (optional)

We do not collect passwords.

## How We Use Information

We use this information only to:
- Identify you as a user
- Associate you with one or more households
- Provide core app functionality

We do not sell user data.
We do not share user data with third parties.

## Data Storage

User data is stored securely using Supabase.
Access is restricted to the minimum required to operate the app.
